- A program lényege, hogy a denevér elkapja a macskát, aki rajzol a pályára. 
    A macska amint a denevér elkapja egy véletlen helyre ugrik és folytatja
    eddigi tevékenységét, amíg újra nem találkozik üldözőjével.
- Külső beavatkozás nem szükséges, kivéve, ha le szeretnénk törölni a pályát,
    ami "SPACE" billentyű lenyomásával lehetséges.
